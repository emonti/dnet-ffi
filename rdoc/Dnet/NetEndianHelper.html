Module: Dnet::NetEndianHelper

A special helper for network packet structures which use big-endian or "network" byte-order. This helper generates read/write accessors that automatically call the appropriate byte conversion function, ntohs/ntohl for ‘reading’ a 16/32 bit field, and htons/htonl for writing to one.

NOTE this helper does not currently do anything special for 64-bit values at all.

This helper relies on a included loaded FFI::DRY::StructHelper module to do its magic. Always include this module as follows: 

  class YourStruct < ::FFI::Struct # or ::FFI::ManagedStruct, (or Union?)
    include ::FFI::DRY::StructHelper
    include ::Dnet::NetEndianHelper
    ...
