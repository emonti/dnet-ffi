Module Dnet::ConstFlagsMap
In: lib/dnet/helpers.rb

Behaves just like ConstFlags, except that the [nnn] returns a list of names for the flags set on nnn. Name string lookups work same way as ConstFlags.

[](arg)

A flexible lookup. Takes ‘arg’ as a Symbol or String as a name to lookup a value, or an Integer to lookup a corresponding names for the flags present in it.

[Source]

# File lib/dnet/helpers.rb, line 310
    def [](arg)
      if arg.is_a? Integer
        ret = []
        list.invert.sort.each {|v,n| ret << n if (v & arg) == v }
        return ret
      elsif arg.is_a? String or arg.is_a? Symbol
        list[arg.to_s.upcase]
      end
    end
