[](arg)

A flexible lookup. Takes ‘arg’ as a Symbol or String as a name to lookup a value, or an Integer to lookup a corresponding name.

    def [](arg)
      if arg.is_a? Integer
        list.invert[arg]
      elsif arg.is_a? String or arg.is_a? Symbol
        list[arg.to_s.upcase]
      end
    end
list()

Generates a hash of all the constant names mapped to value. Usually, it‘s a good idea to override this like so in derived modules:
