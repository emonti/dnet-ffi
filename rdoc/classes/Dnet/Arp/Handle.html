Class Dnet::Arp::Handle
In: lib/dnet/arp.rb

A handle for accessing the kernel arp(4) cache. This does not require root privileges.

get(addr)

Retrieves the ARP entry for the protocol address specified by ‘addr’ (supplied as a String). Uses dnet(3)’s addr_aton() function to parse ‘addr’ and arp_get() to retrieve a result, which is cast as an Entry instance.

[Source]

# File lib/dnet/arp.rb, line 96
      def get(addr)
        _check_open!
        ae = Entry.new
        return ae if ae.pa.set_string(addr) and 
                     ::Dnet.arp_get(@handle, ae) == 0
      end
