Module Dnet::Intf::Entry::Flags
In: lib/dnet/intf.rb

Constants map for interface flags. This is a map of flags, so lookups for integers using [nnn] return a list of flags present in nnn. Lookups for names using ["XX"], ["xx"] or [:xx] still return the value of the constant named XX.
