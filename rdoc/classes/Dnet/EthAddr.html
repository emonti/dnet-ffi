new(*args)

Adds the ability to initialize a new EthAddr with a mac address string such as ‘de:ad:be:ef:ba:be’. This argument is only parsed if it is passed as the only String argument.

[Source]

    def initialize(*args)
      if args.size == 1 and (s=args[0]).is_a? String
        raise "bad mac address" unless s =~ MAC_RX
        raw = ::Dnet::Util.unhexify(s, /[:-]/)
        super(::FFI::MemoryPointer.from_string(raw))
      else
        super(*args)
      end
    end
